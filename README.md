## 소개  
인증, 인가, 회원가입, 로그인 등 웹 로그인에 대한  
보일러 플레이트를 제작하는 프로젝트 입니다.  
  
- 로그인, 회원가입, 자동로그인  
- 카카오 로그인  
- refresh Token 구현을 통한 보안 향상 (서버측에서 User 로그아웃 가능(token을 db에서 삭제))  
- 계정 찾기(구현중..)  

## 업데이트  
- Kakao 로그인 지원  
- 웹 호스팅  

## 기술 스택
**Node.js**  
**Express.js**  
**JsonWebToken**  
**Oauth2.0 with Kakao login API**  
**MySql Database**  
**AWS RDS**  
**AWS Elastic Beanstalk**  
**MVC Pattern**
